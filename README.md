# Остап - ШІ-помічник для людей із вадами зору.

> **Статус: В розробці**

Telegram-бот із використанням великих мовних моделей, розроблений спеціально для людей із вадами зору. Остап розуміє голосові повідомлення, аналізує фотографії та відповідає голосом - без необхідності читати текст.

P.S. За замовчуванням відповідає текстом, але це можна змінити - варто просто його про це попросити. Також варто зазначити, що відповідь не завжди може бути стабільною, ведеться розробка і тести. 

P.S.S Зараз бот в дебаг-моді - розробник зможе бачити ваші діалоги із ботом, майте це на увазі під час спілкування.

Його username в Telegram - @EyeHelpYou_bot

---

## Можливості

- **Голосовий ввід** - розпізнавання мовлення через Whisper (підтримка української мови)
- **Аналіз зображень** - опис фото, розпізнавання об'єктів, тексту, навколишнього середовища
- **Голосові відповіді** - бот може генерувати голосові повідомлення за допомогою синтезу мовлення через Edge TTS (`uk-UA-OstapNeural`)
- **Пошук в інтернеті** - бот сам вирішує коли потрібна актуальна інформація
- **Текстовий режим** - для тих хто віддає перевагу тексту і використанню вбудованих screen reader-ів
- **Пам'ять розмови** - бот пам'ятає контекст попередніх повідомлень
- **Голосове розпізнавання команд** - бот має LLM-модуль для розпізнавання команд, наприклад: змінити ім'я користувача, змінити ім'я бота, змінити тип виводу відповідей. Списаок може бути доповнений в майбутньому.

## Приклади використання

**Аналіз зображення:**
> Користувач надсилає фото упаковки і питає голосом:
> *"Що написано на цій упаковці?"*
>
> Бот відповідає голосом:
> *"На упаковці написано: Молоко ультрапастеризоване, 2.5% жирності, 1 літр. Виробник - Галичина."*

---

**Пошук інформації:**
> *"Яка сьогодні погода в Харкові?"*
>
> *"Сьогодні в Харкові хмарно, температура мінус три градуси, вітер слабкий."*

---

**Виконання команд:**
> *"Називай мене Наталя"*
>
> *"Добре, тепер я буду називати вас Наталя."*

> *"Тепер ти будеш Олег""*
> 
> *"Добре, тепер мене звуть Олег"*
---

**Інструкції з покращення фото** *(якщо фото нечітке)*:
> *"Фото трохи розмите. Спробуйте притримати телефон двома руками і не рухатись під час зйомки."*

## Технічний стек

| Компонент              | Технологія               |
|------------------------|--------------------------|
| Платформа              | Telegram Bot (aiogram 3) |
| LLM                    | OpenRouter               |
| Оркестрація            | LangChain                |
| Розпізнавання мовлення | Groq Whisper 3 large     |
| Синтез мовлення        | Microsoft Edge TTS       |
| Обробка аудіо          | ffmpeg                   |
| Хостинг                | fly.io                   |

> Але моделі легко змінюються в якості потреби

```
Повідомлення користувача
        │
        ▼
    Роутер ──────────────────┐
        │                    │
        ▼                    ▼
   Процесор              Командна нода
  (відповідь)      (Зміна імен + зміна типу виводу)
        │
        ▼
  [пошук в інтернеті]
  [аналіз зображення]
        │
        ▼
  Голосова / текстова відповідь
```

## В планах

- Додати можливість моделі виконувати розрахунки (мовні моделі не вміють рахувати, вони просто вгадують)
- Додати передоробку зображення для збільшення чіткості щоб зменшити кількість невдалих спроб користувача відправити фото
- Покращення оптимізації, масштабованості, результату роботи моделей
- Кнопка фідбеку для відправки "пропозицій та зауважень"
- Підтримка обробки декількох фото одразу (зараз можна лише надіслати їх по черзі)
